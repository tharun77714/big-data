============================================================ 1. PROJECT
OVERVIEW ============================================================

Project Name: Dynamic Surge Pricing E‑Commerce Engine

Goal: Build a real‑time system that automatically adjusts product prices
based on user demand using Big Data technologies.

Core Technologies: - Hadoop (HDFS storage) - Apache Spark (data
processing and ML) - Kafka (event streaming) - Python (data generation
and ML) - React / HTML (UI) - NodeJS / Python API (backend)

Real companies using similar systems: Amazon Uber Airbnb Airlines

The system simulates an e‑commerce store where product prices change
automatically based on demand.

Example: Normal demand → $199 High demand → $230 Low demand → $180

The system reads user behavior data and updates price in real time.

============================================================ 2. PROBLEM
STATEMENT ============================================================

Traditional e‑commerce websites use static pricing.

Example: Product price = $200 It stays $200 until a human changes it.

Problems: 1. Sudden demand spikes cannot be captured 2. Competitor price
changes are ignored 3. Inventory shortages are not handled dynamically
4. Revenue opportunities are lost

Solution: Build a real‑time pricing engine that analyzes:

-   User clicks
-   Add to cart actions
-   Purchases
-   Inventory levels
-   Competitor prices

and automatically adjusts product prices.

============================================================ 6. INSTALL
JAVA ============================================================

sudo apt update sudo apt install openjdk-8-jdk

Check:

java -version

============================================================ 7. INSTALL
HADOOP ============================================================

Download Hadoop

wget https://downloads.apache.org/hadoop/common/hadoop-3.3.6.tar.gz

Extract

tar -xvzf hadoop-3.3.6.tar.gz

Move folder

sudo mv hadoop-3.3.6 /usr/local/hadoop

Add environment variables

nano ~/.bashrc

Add:

export HADOOP_HOME=/usr/local/hadoop export
PATH=PATH:HADOOP_HOME/bin:$HADOOP_HOME/sbin

Apply:

source ~/.bashrc

============================================================ 8.
CONFIGURE HADOOP
============================================================

Edit files:

core-site.xml hdfs-site.xml mapred-site.xml yarn-site.xml

Set namenode path

Create directories:

mkdir ~/hadoopdata

Format HDFS:

hdfs namenode -format

Start Hadoop:

start-dfs.sh start-yarn.sh

Check running services:

jps

Expected:

NameNode DataNode ResourceManager NodeManager

============================================================ 9. INSTALL
SPARK ============================================================

Download Spark

wget
https://downloads.apache.org/spark/spark-3.5.1/spark-3.5.1-bin-hadoop3.tgz

Extract:

tar -xvzf spark-3.5.1-bin-hadoop3.tgz

Move:

sudo mv spark-3.5.1-bin-hadoop3 /usr/local/spark

Add environment variables:

export SPARK_HOME=/usr/local/spark export PATH=PATH:SPARK_HOME/bin

============================================================ 10. CONNECT

============================================================ 13. SPARK
STREAM PROCESSING
============================================================

Spark processes events every 10 seconds.

Calculate:

views add to cart purchases

Conversion rate:

============================================================

Output:
